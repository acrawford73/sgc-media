Category

The Category section defines a new category that your Roku channel will display manually or dynamically. The Content included in a Category is based on either a Playlist or a Query String containing one or multiple Tags. There are three default categories that need to be configured for every Roku channel: "Continue Watching", "Most Popular", and "Recently Added". Each Category is displayed as a separate row to end-users.
