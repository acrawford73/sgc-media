Overview

The purpose of this website is to provide a JSON formatted data feed of media content to be consumed by Roku streaming devices. The website seeks to manage the data feed, media content, and its metadata. It allows you to customize the content feed for your Roku Channel. The content feed is based on the JSON Direct Publisher Feed Specification.
