Overview

This website functions as a Content Management System (CMS). It stores both the media files and metadata, which can be managed through the site. A browsable API is provided to allow for the media content to be searched.

The following media types are supported:

  • Video (MP4, M4V, WMV, MOV)
  • Audio (MP3, M4A, WMA, OGG, FLAC)
  • Images (JPG, PNG, GIF, BMP, DNG)
  • Documents (PDF, TXT, CSV, DOC, DOCX, XLS, XLSX, PPT, PPTX, ODS, ODT, ODP, ODG)

The website also provides a content API formatted for Roku streaming devices. The content feed, media content, and its metadata can be managed and customized for your Roku Channel. The content feed is based on the JSON Direct Publisher Feed Specification.
